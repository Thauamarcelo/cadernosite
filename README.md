# Meu Site Pessoal

Este projeto é um espaço que criei para experimentar, aprender e registrar o que venho estudando em desenvolvimento web.

Ele não nasceu como um produto e nem como um portfólio tradicional. É mais um laboratório pessoal — um lugar onde posso testar ideias, praticar HTML semântico, evoluir no CSS e organizar pequenas descobertas ao longo do caminho.

---

## 🎯 Qual é a proposta?

Criar um espaço simples, mas com intenção.

Aqui você pode encontrar:
- Estudos e experimentos com HTML e CSS
- Estruturas de layout
- Pequenos tutoriais
- Ajustes e evoluções constantes no código

Também uso este projeto para mostrar minha evolução técnica para:
- Entrevistadores
- Amigos da área
- Pessoas curiosas
- Ou qualquer desconhecido que queira ver algo novo

---
